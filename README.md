# Cube Mastery Platform

## Project Overview

Cube Mastery is a modern web platform built to help users learn, practice, and master the Rubik's Cube and other twisty puzzles — with structured lessons, timers, and progress tracking.

---

## Tech Stack

This project is built with:

- **Vite** — fast build tooling and dev server
- **TypeScript** — type-safe JavaScript
- **React** — component-based UI
- **shadcn/ui** — accessible, customizable component library
- **Tailwind CSS** — utility-first styling

---

## Getting Started Locally

Make sure you have **Node.js** and **npm** installed. You can use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage Node versions.

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate into the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The app will be available at `http://localhost:5173` by default, with hot-reloading enabled.

## Deployment

This project can be deployed to any static hosting provider that supports Vite builds:

- **Vercel** — connect your GitHub repo and deploy instantly
- **Netlify** — drag and drop your `dist/` folder or connect via Git
- **Cloudflare Pages** — fast global CDN with Git integration

To build for production:

```sh
npm run build
```

The output will be in the `dist/` folder, ready to deploy.
